# stm32_hand_recognize_system
采用K210 视觉识别模块 + STM32 主控单片机的双机协同架构，实现手势识别与舵机智能控制功能。K210 作为视觉感知单元，通过内置 AI 视觉算法实时采集图像画面，精准识别预先训练设定的5 种不同手势动作。当检测到有效手势后，K210 会将不同手势对应的独立识别指令，通过串口通信（USART） 实时发送至 STM32 单片机。 STM32 单片机持续接收并解析 K210 传输的手势数据，对指令进行校验与匹配。针对五种不同的手势信号，STM32 分别输出对应参数的 PWM 控制信号，驱动舵机执行不同角度的旋转动作，完成手势指令与舵机运动的一一对应控制。整套系统实现了从视觉手势识别 — 数据串口传输 — 单片机解析处理 — 舵机精准执行，实现了非接触式的智能机械控制效果。
